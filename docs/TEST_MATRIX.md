# Test Matrix

## Устройства и браузеры

- Android: Chrome, Яндекс Браузер, мобильное приложение Яндекса.
- iPhone: Safari, Chrome, мобильное приложение Яндекса.
- Локальная разработка: desktop Chrome с mobile viewport и touch emulation.

## Размеры экранов

- 360x640 — малый Android.
- 375x667 — iPhone SE.
- 390x844 — современный iPhone.
- 412x915 — крупный Android.
- 430x932 — крупный iPhone.
- Landscape варианты для проверки заглушки поворота.

## Ключевые сценарии

- Запуск приложения.
- Создание Babylon.js сцены.
- Portrait: игра видна и активна.
- Landscape: показана просьба повернуть устройство, gameplay paused.
- Возврат в portrait: gameplay может восстановиться.
- Resize и dynamic viewport.
- Safe area не перекрывает UI.
- visibilitychange/blur/focus ставят игру на паузу и возвращают.
- Touchcancel не ломает состояние.
- Нет системного скролла, swipe-to-refresh, выделения, контекстного меню, случайного zoom.
- Нет накопления обработчиков.

## Критерии прохождения

- Нет Blocker и Critical дефектов.
- Нет необработанных ошибок консоли.
- Сборка проходит и размер ниже 100 МБ.
- Поведение одинаково для одинаковой ситуации.
- Прогресс и награды не дублируются в будущих этапах.
