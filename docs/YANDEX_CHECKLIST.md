# Yandex Games Checklist

Актуальная сверка: 2026-07-17. Основной источник: https://yandex.ru/dev/games/doc/ru/concepts/requirements

## Технические требования

- [ ] SDK Яндекс Игр встроен только через `src/platform/yandex` и `PlatformAdapter`.
- [ ] Авторизация только после осознанного нажатия кнопки.
- [ ] Гостевой режим доступен, внутренний прогресс сохраняется.
- [ ] Звук и игра останавливаются при сворачивании, рекламе и паузе.
- [ ] Платежи, если появятся, только через SDK Яндекс Игр.
- [ ] Для mobile-only выбран портретный режим; в landscape показывается заглушка поворота.
- [ ] Игра полностью управляется жестами, mouse только для разработки.
- [ ] Long tap не вызывает выделение или контекстное меню.
- [ ] Прогресс сохраняется сразу после значимого действия.
- [ ] Нет системного скролла и swipe-to-refresh.
- [ ] `LoadingAPI.ready()` вызывается после полной загрузки и готовности пользователя играть.
- [ ] `GameplayAPI.start()`/`stop()` соответствуют реальным моментам геймплея.
- [ ] Поддерживаются Android 5+ и iOS 9+ как минимальная платформа требований.
- [ ] Размер всех файлов не превышает 100 МБ; цель проекта — до 50 МБ.
- [ ] `index.html` находится в корне архива.
- [ ] В именах файлов и папок нет пробелов и кириллицы.
- [ ] Не используется интерактивный ИИ в gameplay.

## Пользовательский опыт и контент

- [ ] Есть описание управления.
- [ ] Есть прогресс, сюжет/сеттинг и нарастающая сложность.
- [ ] Основное содержание рассчитано более чем на 10 минут: 30 уровней по 20–60 секунд с повторами и заданиями.
- [ ] RU и EN локализация.
- [ ] Все материалы оригинальные или лицензированные.
- [ ] Игра не является копией другой игры.
- [ ] Реклама только через SDK, в логических паузах, без имитации рекламных блоков.

## Статус этапа 1

На этапе 1 реализован только mock-адаптер. Реальный SDK будет подключён на этапе 8.

## Stage 2 update

- Интерактивный gameplay остаётся внутри Babylon/Vite приложения; прямых обращений к Yandex SDK из gameplay-модулей нет.
- Pause/resume при landscape, blur и visibility сохраняются; реклама и rewarded-награды не добавлялись на этапе 2.
- `index.html` остаётся в корне, имена новых файлов — латиницей без пробелов и кириллицы.
- Требования Яндекс Игр сверялись с официальной страницей требований 2026-07-17: https://yandex.ru/dev/games/doc/ru/concepts/requirements
