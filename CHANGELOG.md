# Changelog

## 0.1.0 — 2026-07-17

### Added

- Mobile-only project documentation in `docs/`.
- Vite + strict TypeScript project skeleton.
- Babylon.js scene with camera, light, and test object.
- Portrait adaptive viewport, safe area support, and rotate-device overlay.
- Scroll, accidental zoom, text selection, and context menu blocking.
- Platform adapter interface and local mock adapter.
- Basic lifecycle handling for pause, focus, visibility, resize, and orientation change.
- Vitest and Playwright smoke coverage.
