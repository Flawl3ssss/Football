# Changelog

## 0.1.0 — 2026-07-17

### Added

- Документация этапа 1 и scope первой версии.
- Аудит требований mobile-only и Яндекс Игр.
- TypeScript/Vite/Babylon.js технический каркас.
- Мобильная оболочка portrait 9:16 с landscape-заглушкой.
- MockPlatformAdapter и базовый GameState.
- Unit-тесты и Playwright smoke-тесты.

### Not Added

- Футбольная механика, мяч, пас, удар и уровни не реализованы на этапе 1.

## 0.2.0 — Stage 2 prototype

- Добавлен технический прототип траектории мяча: поле, ворота, стойки, перекладина, временная сетка, мяч, мобильная камера, preview траектории и детерминированная симуляция исходов.
- Добавлен единый PointerInput для touch-first управления с mouse только для локальной разработки, защитой от лишних пальцев, `pointercancel`/`touchcancel`, blur и visibility cancellation.
- Добавлены временные оригинальные low-poly футболисты двух команд, вратарь, state machine интерактивного эпизода и автоматическое разыгрывание без полноценного футбольного симулятора.
- Удалён бинарный smoke-скриншот из репозитория, чтобы PR можно было сохранить в системах без поддержки бинарных файлов.
