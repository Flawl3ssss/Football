# Карьерный путь: футбольная дуга

> Рабочее название временное. Перед публикацией название нужно проверить на уникальность в каталоге Яндекс Игр и поиске.

Мобильная low-poly 3D-игра о футбольной карьере для Яндекс Игр. Игрок одной рукой рисует пальцем траекторию будущего паса или удара, после чего короткий футбольный эпизод автоматически разыгрывается. На этапе 1 футбольная механика не реализуется: репозиторий содержит документацию, технический каркас, мобильную оболочку и тестовый 3D-объект.

## Scope первой версии

- Только mobile-only: Android, iPhone, мобильные браузеры и мобильное приложение Яндекса.
- Основная ориентация — портретная, базовая композиция — 9:16.
- Управление одной рукой; основной ввод — touch, mouse разрешён только для локальной разработки.
- 30 уровней, 3 главы карьеры, уровни по 20–60 секунд и 1–4 решения.
- Около 15 мячей, около 6 комплектов формы, около 6 эффектов гола.
- Монеты, косметический магазин, задания, достижения, ежедневное испытание, лидерборды.
- Русский и английский языки.
- Локальные и облачные сохранения с версиями и миграциями.
- Релевантные функции Yandex Games SDK через PlatformAdapter.

## Принципы оригинальности и честной монетизации

- Игра не должна копировать название, интерфейс, ассеты, уровни, персонажей, модели, форму, тексты, звуки или визуальную идентичность референсов.
- Все модели, интерфейс, уровни, звуки и визуальные материалы должны быть оригинальными или иметь подходящую лицензию.
- Косметические предметы не дают игровых преимуществ.
- В первой версии нет лутбоксов, энергии, обязательной рекламы, тёмных паттернов, полноценного футбольного симулятора и синхронного мультиплеера.

## Стек

- TypeScript strict
- Babylon.js
- Vite
- Vitest
- Playwright
- ESLint
- Prettier
- glTF/GLB для будущих ассетов
- Yandex Games SDK через PlatformAdapter
- MockPlatformAdapter для локальной разработки

## Структура

```text
src/game/core        # запуск, сцена, состояние, lifecycle
src/game/gameplay    # будущая футбольная логика, пока без реализации
src/game/levels      # будущие data-driven уровни
src/game/entities    # будущие игровые сущности
src/game/animation   # будущие анимации
src/game/camera      # камера и композиция
src/ui               # мобильная оболочка и подсказки
src/economy          # будущий баланс монет и магазина
src/progression      # будущие сохранения, задания, достижения
src/services         # сервисы viewport/lifecycle
src/platform         # PlatformAdapter
src/platform/mock    # локальный mock без SDK
src/platform/yandex  # будущая интеграция SDK
src/localization     # RU/EN словари
tests                # unit и e2e тесты
```
